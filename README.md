# 🛡️ FortressPass

### Advanced Password Security & Strength Analysis Platform

FortressPass is a modern cybersecurity application developed as part of my Cyber Security Internship at Prodigy InfoTech.

The platform analyzes password strength in real time, identifies security weaknesses, and provides intelligent recommendations to help users create stronger and more secure passwords.

---

## 🚀 Features

### Password Strength Analysis

* Real-Time Password Evaluation
* Security Score (0–100)
* Weak, Medium, Strong Classification
* Password Complexity Analysis
* Instant Security Feedback

### Security Checks

* Minimum Length Validation
* Uppercase Letter Detection
* Lowercase Letter Detection
* Number Detection
* Special Character Detection
* Password Pattern Analysis

### Additional Features

* Password Generator
* Password Visibility Toggle
* Copy Password Functionality
* Security Recommendations Panel
* Password History Tracking
* Modern Cyber Security Dashboard
* Fully Responsive Design

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Aathekesavan/PRODIGY_CS_03.git
```

Navigate into the project:

```bash
cd PRODIGY_CS_03
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run dev
```
